== Description ==

Razor is a sophisticated cross-platform unit-test framework that performs automated unit tests on PHP and JavaScript code for WordPress plugins. 

<h4>Cross Platform</h4>

<h4>No Dependencies</h4>
Razor contains specially modified forks of PHPUnit and JSUnit that let it spin-up instantly on ANY server, without frustrating PEAR dependencies and time consuming setup tasks.
