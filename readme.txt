State of the art cross-platform automated unit testing for WordPress plugins, on Linux, Windows, and Mac OS.

== Description ==

Razor is a sophisticated cross-platform unit testing framework that checks PHP and JavaScript code used in WordPress plugins. 

<h4>Cross Platform</h4>
Razor works on ANY version of Linux, ANY version of Windows (from XP to 7) and all current versions of Mac OSX. It's designed to be checked-in 

<h4>No Dependencies</h4>
Razor contains specially modified forks of PHPUnit and JSUnit that let it spin-up instantly on ANY server, without frustrating PEAR dependencies and time consuming setup tasks.
